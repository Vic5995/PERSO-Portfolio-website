Todo App

Mobile Project

The design of this application was found on Figma.

This application has been a real learning experience. I was able to review what I already knew about the ROOM library for data management in an application.

I discovered the LiveData and the ViewModel to transmit information between the different parts of the application.

For the moment, the application allows saving tasks and to find them when we come back to the application (thanks to data persistence). You can mark it as completed, and you can also delete a task.

For the future, I'm thinking of enriching the tasks model by adding tags to sort the list of tasks. An interesting post would be to add subtasks.

There is also some work to do on the UI, for the deletion of a task, for the moment I choose a button, but it is more natural to use a swipe to delete or archive a task.

Then, connecting to a remote database can be a challenge as well as sending notifications when the deadline approaches.
